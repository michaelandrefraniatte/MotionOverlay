Motion Capture as Overlay in Games

Overlay in Your Games with Your Gaming Device Motions

The program enable an overlay for showing gameplay with gaming devices in action, while recording or streaming. Let's see...

Considerations About the Program

The Numpad keys are allowing to relocate position of overlay, change the size, the transparency, and even make disapearing the overlay. It only run with windowed mode of your game, not fullscreen (toggle with Alt+Enter). To note, there's file for parameters. I also made an overlay for the webcam. To note, there's file for borders. You can change the border by changing the gif file coming with it. It can be a gif file animated. It show also audio visualization coming in your default sound card. The numpad keys 0-9 are used for change position of Webcam capture, the numpad key multiply for toggle half transparency, and the numpad key minus for toggle display. For games in windowed mode but still displaying a border at top, I made a tool to remove it. To note, the page up and page down keys are used to remove/add the border at top.
